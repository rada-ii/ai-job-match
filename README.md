# AI Job Match Agent

> Intelligent job matching powered by AI — analyzes your profile, finds relevant positions, and generates personalized cover letters.

## About

AI Job Match Agent is a full-stack application that uses Claude API to match job seekers with relevant positions. Upload your CV or fill in your profile, set your preferences, and the agent will analyze available listings, rank them by relevance, and generate tailored cover letters for your top matches.

Built as a demonstration of how AI can enhance recruitment workflows — making job search smarter, faster, and more personalized.

## Planned Features

- **Profile Analysis** - AI parses your CV/profile to understand skills, experience, and preferences
- **Smart Matching** - Claude API analyzes job listings and ranks them by relevance to your profile
- **Cover Letter Generation** - Personalized cover letters generated for each matched position
- **Dashboard** - Track your matches, saved jobs, and generated letters
- **Filter & Search** - Filter by location, salary range, remote/onsite, tech stack
